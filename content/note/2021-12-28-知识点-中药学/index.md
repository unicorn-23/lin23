# 绪论

### 什么是中药？

在中医理论指导下，用于预防、治疗、诊断疾病并具有康复与保健作用的物质。

### 特点

1.  得天独厚的自然资源（植物药11140多种、动物药1580、矿物80多种）
2.  历史悠久，反映了我国历史文化背景
3.  独特的理论体系
4.  特有的应用形式
5.  以中国传统医药理论指导采集、炮制、制剂，说明作用机理，指导临床应用

### 什么是本草？

-   自古相沿把中药学称为本草
-   “诸药以草为本”（草：植物药）
-   韩保昇：“药有玉石草木虫兽，而直言本草者，草类药为最多也。”

### 什么是中药学？

是专门研究中药基本理论和各种中药的品种来源、产地、采集、炮制、**性能、功效、临床应用规律**等知识的一门学科。

# 第二章 中药的产地和采集

### 中药的来源

主要是天然的植物、动物和矿物

《用药法象》：“凡诸草木昆虫，产之有地：根叶花实，采之有时。失其地则性味少异，失其时则性味不全。”

### 原则

保证质量、保护药源

### 产地

-   自然条件不同（气候地貌、水土日照）
-   自然地理状况复杂，生态环境差异，生物分布不同，质量优劣有别
-   中药材品种、产量、质量有明显的地域性

### 道地药材

优质纯真药材的专用名词

-   历史悠久
-   炮制考究
-   产地适宜
-   疗效突出
-   品种优良
-   有地域特点
-   产量宏丰

云南-三七、茯苓

四川-黄连、川芎、川贝母、乌头

东北-人参、细辛、五味子

河南-地黄、牛膝、山药、菊花（四大怀药）

宁夏-枸杞

甘肃-当归

山东-阿胶

江苏-薄荷

### 采集

-   依时采集：中药的采收时节和方法对确保药物的质量有密切的关联
-   不同时期、药用部分所含有效及有害成分各不相同（药物的疗效和毒副作用也往往有较大差异）
-   孙思邈：“早则药势未成，晚则盛时已歇。”

全草：大多数在植物枝叶繁茂、花朵初开时采集。

叶类：通常在花蕾将放或正盛开的时候，此时叶片茂盛。

花类：一般采收未开放的花蕾或刚开放的花朵，以花粉入药者须在花朵盛开时采取。

果实、种子 ：一般在果实成熟时采收，少数药材要在果实未成熟时采收果皮或过失

根、根茎：一般以秋末或春初即二月、八月采收，“春宁宜早，秋宁宜晚。”

树皮、根皮：通常在春夏时节植物生产旺盛，植物体内浆液充沛时采集。

虫类：多在夏末秋初捕捉

贝壳类：多在夏秋季捕采

大动物类药材：四季皆可，宜在秋季猎取

# 第三章 中药的炮制

### 炮制含义，增效减毒

指药物在应用或制成各种剂型前，根据医疗、调制、制剂的需要，而进行必要的加工处理的过程，是我国的一项传统制药技术。

古时称为“炮炙”、“修事”、“修治”

### 意义

-   符合临床用药需要，保障药效
-   适于不同的药性和治疗要求
-   确保有毒之品的用药安全

### 饮片

以中药材为原料，按中药炮制方法，经过加工工序，制成片、段、丝、块等一定规格，可直接用于中医临床的中药炮制品。 

### 炮制的目的

1.  纯净药材，保证质量，分捡药物，区分等级
2.  切制饮片，便于调剂制剂
3.  干燥药材，利于贮藏
4.  矫味、矫臭，便于服用
5.  降低毒副作用，保证安全用药
6.  增强药物功能，提高临床疗效
7.  改变药物性能，扩大应用范围
8.  引药入经，便于定向用药

### 炮制的方法

1.  修治

    1.  纯净药材
    2.  粉碎药材
    3.  切制药材

2.  水制：用水或其他辅料处理药材的方法

    1.  漂洗
    2.  浸泡
    3.  闷润
    4.  喷洒
    5.  水飞：借药物在水中的沉降性质分取药材极细粉末的方法（常用矿物类、甲壳类药物的治粉，如朱砂、炉甘石、滑石、蛤粉、雄黄）

3.  火制：经过火加热处理的方法

    根据加热的温度、时间、方法的不同，可分为：

    1.  炒（与固体辅料）：根据火候可分为
        1.  炒黄
        2.  炒焦
        3.  炒炭：能缓和药物的烈性或副作用，或增强其收敛止血、止泻的作用（炒炭存性）
    2.  炙（与液体辅料）：将药物与液体辅料共置锅中加热拌炒，使辅料渗入药物组织内部或附着于药物表面，以**改变药性，增强疗效或降低毒副作用**的方法称炙法。
    3.  煅
        1.  明煅：让矿石高温后变酥脆
        2.  闷煅
    4.  烫：先将辅料加热，温度升高后，将需要炮制的药物迅速扔到辅料中。
    5.  煨：药物外边包裹湿纸或湿面，然后加热。

4.  水火共制

    1.  蒸
    2.  煮
    3.  炖
    4.  潬：焯水
    5.  淬：煅淬，淬火

5.  其他制法

    1.  制霜
        1.  榨去油质的残渣
        2.  煮后剩下的残渣
        3.  重新结晶
    2.  发酵
    3.  精制
    4.  药拌
    5.  发芽

# 第四章 药性理论

### 药物防治疾病的基本作用

扶正祛邪，消除病因，恢复脏腑的正常生理功能，纠正阴阳气血偏盛偏衰的病理现象，使之最大程度上恢复到正常状态，达到治愈疾病，恢复健康的目的。

### 药物的偏性

各种药物本身各自具有若干特性和作用

以药物的偏性来纠正疾病所表现出来的阴阳偏盛偏衰（以偏纠偏）

### 药性

-   把药物与疗效有关的性质和性能统称为药性
-   药物发挥疗效的物质基础
-   治疗过程中所体现出来的作用
-   是药物性质与功能的高度概括

### 药性理论

**含义**：研究药性形成的机制及其运用规律的理论称为药性理论，其基本内容包括四气五味、升降浮沉、归经、毒性。

**依据**：阴阳、脏腑、经络学说

**实际意义**：用药规律

## 第一节 四气

### 四气的含义

四气又称四性，即**寒、热、温、凉**四种药性，是药物性能的重要标志之一。

反映了药物对人体阴阳盛衰、寒热变化的作用倾向。

**平性**：是指寒热界限不很明显、药性平和、作用较缓和的一类药。

### 四气的形成

**确定依据：**药物作用于人体所产生的不同反应，所获的的不同疗效

**结论：**与所治疗疾病的性质是相对而言，能够减轻或消除热症的药物，一般属寒性或凉性，反之亦然。

### 四气的作用（四气与功效的关系）

**寒凉药：**清热泻火、凉血解毒、滋阴除蒸、泻热通便、清热利尿、清化热痰、清心开窍、凉肝息风。

**温热药：**温里散寒、暖肝散结、补火助阳、温阳利水、温经通络、引火归原、回阳救逆。

### 四气对临床用药的指导意义

寒者热之，热者寒之

疗寒以热药，疗热以寒药

**用药原则**

寒凉药用治阳热证

温热药用治阴寒证

### 四气与阴阳

寒凉属阴，温热属阳

寒凉与温热时相对立的两种药性

寒凉之间、温热之间则仅是程度上的不同

凉次于寒，温次于热

## 第二节 五味

### 五味的含义

药物有**酸、苦、甘、辛、咸**、淡、涩，五种（七种）不同的味道，并具有不同的治疗作用。

### 五味的形成

基于口尝，反映药物真实味道。

不同味道的药物作用于人体，不同的反应和疗效，对药物作用进行高度概括

### 五味的阴阳属性

《内经》：“辛甘淡属阳、酸苦咸属阴。”

### 五味的主要作用及主治病证

-   辛：
    -   作用：“能散能行“，发散邪气、行气行血
    -   主治：表证及气血阻滞之证
    -   举例：解表药、行气药、活血药
    -   补充：芳香药性，辟秽解表、运脾化湿、通窍止痛、行气活血、开窍醒神
-   酸：
    -   作用：“能收能涩”，收敛、固涩
    -   主治：体虚滑脱证（防止人体气血津液滑脱）
    -   举例：固表止汗、敛肺止咳、涩肠止泻、固精缩尿、固崩止带等药物
-   甘：
    -   作用：“能补能和能缓”，补益、和中（脾胃）、调和药性、缓急止痛
    -   主治：正气虚弱、身体诸痛
    -   用于：调和药性、中毒解救
    -   举例：滋养补虚、调和药性及制止疼痛的药物
-   苦：
    -   作用及主治：
    -   能泻：
        -   清泻火热：火热证
        -   泻降气逆：喘咳、呕恶
        -   通泄大便：便秘
    -   能燥：燥湿邪：湿证（寒湿或湿热）
    -   能坚：坚阴（泻火存阴）：阴虚火旺证
    -   举例：清热泻火、下气平喘、降逆止呕、通利大便、清热燥湿、苦温燥湿、泻火存阴
-   咸：
    -   作用：“能下、能软”，泻下通便、软坚散结
    -   主治：大便燥结、瘰疬痰核、瘿瘤、癥瘕痞块
    -   举例：泻下药或润下通便药，软化坚硬、消散结块的药物
    -   咸走血及咸入肾
-   淡：
    -   作用：“能渗、能利”，渗湿利小便
    -   主治：水肿、脚气、小便不利之证
    -   举例：利水渗湿的药物具有淡味
-   涩：
    -   作用：与酸味作用相似
    -   主治：虚汗、泄泻、尿频、遗精、滑精、出血

### 性味关系

性和味分别从不同角度说明药物的作用性质和特点

四气主要概括药物影响人体寒热变化的作用性质

五味提示药物多方面的基本作用

四气和五味结合，准确的辨别药物的作用

复方用药，或用其气或用其味

### 性味结合规律

-   气味相同，作用相近（辛温的药物多具有发散风寒的作用）
-   气味不同，作用有别
    -   气同味异（麻黄、乌梅、肉苁蓉同属温性）
        -   麻黄**辛温**散寒解表
        -   乌梅**酸温**敛肺涩肠
        -   肉苁蓉**咸温**补肾阳
    -   味同气异（薄荷、附子、石膏均为辛味）
        -   薄荷**辛凉**疏散风热
        -   附子**辛热**补火助阳
        -   石膏**辛寒**清热降火
-   一药兼有数味，治疗范围广（多功效性）
    -   当归辛甘温，甘以补血、辛以活血行气、温以祛寒
    -   可用治血虚、血滞、血寒引起的多种疾病

## 第三节 升降浮沉

### 升降浮沉的含义

是指药物对机体有向上、向下、向外、向内四种不同作用趋向

-   升：上升提举，趋向于上
-   降：下达降逆，趋向于下
-   浮：向外发散，趋向于外
-   沉：向内收敛，趋向于内

### 升降浮沉的形成

-   通过药物作用于机体所产生的不同疗效
-   药物表现出的不同作用趋向
-   是与疾病所表现的趋向性相对而言的

| 病势上           | 病位上           |
| ---------------- | ---------------- |
| 向上（如呕吐）   | 在表（如表证）   |
| 向下（如脱肛）   | 在里（如便秘）   |
| 向外（如自汗）   | 在上（目赤肿痛） |
| 向内（表证入里） | 在下（腹水尿闭） |

改善或消除这些病证的药物，相对来说分别具有升降浮沉的作用趋向

###  升降浮沉的作用

-   升浮药：性温热，味辛、甘、淡，质轻，作用趋向多主上升、向外
    -   作用：疏散解表、行气活血、开窍醒神、升阳举陷
-   沉降药：性寒凉，味酸、苦、咸，质量坚实，作用趋向多主下行向内
    -   作用：清热泻火、泻下通便、利水渗湿等

### 升降浮沉的临床意义

1.  调整脏腑气机的紊乱，使之恢复正常的生理功能
2.  作用于机体的不同部位，因势利导，驱邪外出

1.  病势：上逆者宜降，下陷者宜升
    -   肝阳上亢、头晕目眩应选用代赭石、石决明等沉降药来平肝潜阳
    -   气虚下陷、久泻脱肛应选用黄芪、升麻、柴胡等升浮药来升阳举陷
2.  病位：在下在里者宜沉降，在上在表者宜升浮 

### 药物性味与升降浮沉

凡味属辛、甘，气属温、热的药物，多升浮药

凡味属苦、酸、咸，性属寒、凉的药物，多沉降药

### 药物质地与升降浮沉

一般来说

花、叶、皮、枝等质轻的药物多为升浮药

种子、果实、矿物、贝壳及质重者大多为沉降药

特殊：**诸花皆升，旋覆独降，诸子皆降，苍耳独升**

### 药物炮制与升降浮沉

通过炮制改变趋向

-   酒制则升
-   姜炒则散
-   盐炒下行
-   醋炒收敛

###   配伍与升降浮沉

药物的升降浮沉通过配伍可以受到抑制

升浮药在大队沉降药中能随之下降

沉降药在大队升浮药中能随之上升

###  注意

部分药物本身就具有双向性

升降在物，亦在人也

## 第四节 归经

### 归经的含义

归经是指药物对于机体某部分的选择性作用，即某药对某些脏腑经络有特殊的亲和作用，因而对这些部位的病变起着主要或特殊的治疗作用

### 归经意义

药物治病的使用范围，药效所在，包含了药物定性定位的概念

### 归经理论的形成

先声：《内经》《神农本草经》《名医别录》

基础：《伤寒论》，**六经分经用药**

雏形：唐宋时期，与脏腑经络联系《食疗本草》

确立：金元时代，纳入药性主要内容《珍珠囊》

完善：清代采用五脏六腑之名，说明药性；产生了**卫气营血**及**三蕉归经**的新概念

### 归经的方法

1.  脏腑辨证用药
2.  六经用药
3.  卫气营血用药
4.  三蕉用药

**脏腑经络学说**为归经的理论基础，归经的实质是脏腑经络学说

### 归经理论的依据

-   脏腑经络生理特点：内在脏腑病变可反映到体表上来
-   临床用药经验：胸闷喘咳，以苏子治愈，说明苏子归肺经
-   中医辨证理论体系的发展完善：相应出现六经卫气营血、三蕉用药的归经方法
-   药物自身的特性：以形、气、五味、禀赋归经

### 归经对临床的指导意义

1.  直到临床辨证用药

    -   根据临床表现
    -   辨证审证求因
    -   诊断病变所在的脏腑经络部位
    -   按照归经选择用药
    -   如：肺热咳喘，清热药，桑白皮
    -   胃火牙痛，清胃热，黄连
    -   一药能归数经，其治疗范围扩大

2.  区别功能相似的药物

    以归经的差异选用相同功效的药物，有助于提高用药的准确性

    黄芩、黄连、黄柏：苦寒，沉降，清热燥湿泻火

    区别：

    -   黄芩：上焦湿热，肺热
    -   黄连：中焦湿热，心火，胃热
    -   黄柏：下焦湿热，肾火

3.  注意脏腑病变的相互影响

    培土生金：补脾益肺

    水不涵木：益阴降火

### 归经与性味综合运用

把药物的治疗作用与病变所在的脏腑经络部位有机地联系起来

-   四气：药物的寒热属性
-   五味：具体治疗作用
-   升降浮沉：药物的定向概念
-   归经理论：药物的定位概念

### 肺

-   紫苏：温散风寒
-   薄荷：凉散风热
-   干姜：温肺化饮
-   黄芩：性寒清肺

## 第五节 毒性

###  古代药物毒性的含义较广

毒药是药物的总称

毒性是药物的偏性

毒性是药物毒副作用大小的标志

明代张景岳《类经》：“药以治病，因毒为能，所谓毒者，因气味之偏也。...大凡可辟邪安正者，均可称为毒药，故曰毒药攻邪也。”

### 现代药物毒性的概念

毒性：一般指药物对机体所产生的不良影响及损害性

毒药：一般指对机体发生化学或物理作用，能损害机体引起功能障碍疾病甚至死亡的物质

剧毒药：

-   中毒剂量与治疗剂量比较接近，或某些治疗量已达到中毒剂量的范围，安全系数小。
-   是指毒性对机体组织器官损害剧烈，可产生严重或不可逆的后果。

副作用：是指在常用剂量时出现与治疗需要无关的不适反应，一般比较轻微，对机体危害不大，停药后可自行消失。（与药物自身特性、炮制、配伍、制剂等多种因素有关）

过敏反应：也属于不良反应范畴，其症状轻者可见瘙痒、皮疹、胸闷、气急，重者可引起过敏性休克（除药物因素外，多与患者体质有关）

### 中药毒性分级

大毒、有毒、小毒

### 如何正确对待中药毒性

1.  总体评价-安全有效
2.  本草文献记载-去伪存真
3.  临床报道-借鉴经验
4.  使用管理-遵循法规

### 有毒中药

列入国务院《医疗用毒性药品管理办法》的中药品种（28种）

砒石、砒霜、水银、生马钱子、生川乌、生草乌、生白附子、生附子、生半夏、生南星、生巴豆、斑蝥、青娘虫、红娘虫、生甘遂、生狼毒、生藤黄、生千金子、生天仙子、闹羊花、雪上一枝蒿、红升丹、白降丹、蟾酥、洋金花、红粉、轻粉、雄黄。

### 产生中药中毒的主要原因

1.  剂量过大
2.  误服伪品
3.  炮制不当
4.  制剂服法不当
5.  配伍不当
6.  药不对证
7.  个体差异

### 指导临床用药的意义

1.  应用毒药时，中病即止

    注意体质、疾病、药物、计量、禁忌、炮制、剂型等

2.  以毒攻毒，发挥毒药治疗作用

3.  中毒后及时救治
